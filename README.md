# Turkey Capital Markets

Analysis of Turkish capital markets across **fixed income, derivatives, and equities**, built using live market data during a summer rotation at Borsa İstanbul (Jun–Jul 2026).

Each project applies a standard markets-analysis framework to Turkish data — an emerging market with unusually rich macro dynamics (high inflation, an active central bank, and a volatile currency) that make it a compelling case study for how prices respond to policy and risk.
